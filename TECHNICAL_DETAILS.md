(Unsupervised) لتقسيم البيانات إلى K مجموعات متجانسة.\n\n### خطوات الخوارزمية | Algorithm Steps\n\n```\n1. التهيئة (Initialization):\n   - اختر K مراكز عشوائية\n   - في حالتنا: K = 10 (للأرقام 0-9)\n\n2. التعيين (Assignment):\n   - لكل نقطة بيانات، احسب المسافة إلى كل مركز\n   - عين النقطة إلى أقرب مركز\n\n3. التحديث (Update):\n   - احسب متوسط كل مجموعة\n   - حدّث مراكز المجموعات\n\n4. التقارب (Convergence):\n   - كرر الخطوات 2-3 حتى لا تتغير المراكز\n```\n\n### الصيغة الرياضية | Mathematical Formulas\n\n#### حساب المسافة (Euclidean Distance):\n```\nd(x, c) = √(Σ(x_i - c_i)²)\n\nحيث:\nx = نقطة البيانات\nc = مركز المجموعة\n```\n\n#### تحديث المركز:\n```\nc_k = (1/|C_k|) * Σ(x ∈ C_k) x\n\nحيث:\nc_k = المركز الجديد للمجموعة k\n|C_k| = عدد النقاط في المجموعة k\n```\n\n#### دالة الخسارة (Inertia):\n```\nJ = Σ Σ ||x - c_k||²\n    k x∈C_k\n\nحيث:\nJ = مجموع المسافات المربعة\n```\n\n### التعقيد الزمني | Time Complexity\n\n```\nالتعقيد الزمني: O(n * k * i * d)\n\nحيث:\nn = عدد نقاط البيانات\nk = عدد المجموعات\ni = عدد التكرارات\nd = عدد الأبعاد (الميزات)\n\nفي حالتنا:\nn = 1797 (عينة)\nk = 10 (مجموعات)\ni ≈ 12 (تكرارات)\nd = 64 (ميزات)\n\nالتعقيد الكلي ≈ O(1797 * 10 * 12 * 64) ≈ O(13.8 مليون)\n```\n\n### المميزات والعيوب | Advantages and Disadvantages\n\n| المميزات | العيوب |\n|---------|-------|\n| سريع وفعال | يحتاج لتحديد K مسبقًا |\n| سهل الفهم المستخدمة | Method Used\n\n```python\ndef calculate_confidence(image_scaled, cluster_center):\n    # حساب المسافة\n    distance = np.linalg.norm(image_scaled - cluster_center)\n    \n    # تحويل المسافة إلى ثقة\n    confidence = 1 / (1 + distance)\n    \n    return confidence\n```\n\n### الصيغة الرياضية | Mathematical Formula\n\n```\nConfidence = 1 / (1 + distance)\n\nحيث:\ndistance = ||x - c_k||\nx = الصورة المدخلة\nc_k = مركز المجموعة\n```\n\n### التفسير | Interpretation\n\n```\nDistance = 0   → Confidence = 1.0   (100%) - متأكد جدًا\nDistance = 0.5 → Confidence = 0.67  (67%)  - متأكد نسبيًا\nDistance = 1.0 → Confidence = 0.5   (50%)  - غير متأكد\nDistance = 2.0 → Confidence = 0.33  (33%)  - غير متأكد جدًا\n```\n\n---\n\n## 📐 الصيغ الرياضية | Mathematical Formulas\n\n### 1. تطبيع البيانات | Data Normalization\n\n```\nz = (x - μ) / σ\n\nحيث:\nz = القيمة المطبعة\nx = القيمة الأصلية\nμ = المتوسط\nσ = الانحراف المعياري\n```\n\n### 2. دقة التصنيف | Classification Accuracy\n\n```\nAccuracy = (عدد التنبؤات الصحيحة) / (إجمالي التنبؤات)\n         = (TP + TN) / (TP + TN + FP + FN)\n\nحيث:\nTP = True Positives (إيجابيات صحيحة)\nTN = True Negatives (سلبيات صحيحة)\nFP = False Positives (إيجابيات خاطئة)\nFN = False Negatives (سلبيات خاطئة)\n```\n\n### 3. المسافة الإقليدية | Euclidean Distance\n\n```\nd = √(Σ(a_i - b_i)²)\n\nحيث:\na, b = متجهات البيانات\ni = الفهرس\n```\n\n---\n\n## ⏱️ تعقيد الخوارزميات | Algorithm Complexity\n\n### K-means Complexity\n\n| العملية | التعقيد الزمني | التعقيد المكاني |\n|--------|---|---|\n| التهيئة | O(k*d) | O(k*d) |\n|
